# toolbox-ui
Toolbox UI
## Building
### Development Build
```bash
make
```
### Release Build
```bash
make opt
```
## Running Locally
```bash
open docs/index.html
```
## Publishing
```bash
make opt
git add docs src
git commit -m "description my changes"
git push
```
* Merge changes to main branch.
* Open https://wongphu.github.io/toolbox-ui
